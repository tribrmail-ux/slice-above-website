# Slice Above Woodfired Pizza — Website

The live website for Slice Above, Elisenheim Urban Village, Windhoek.

## Files
- `index.html` — the complete website (self-contained; all fonts & images baked in)
- `og-image.jpg` — link-preview image shown when the site is shared on WhatsApp / Facebook
- `favicon.png` — the little logo shown in the browser tab

## How it's hosted
Deployed on **Cloudflare Pages**, connected to this GitHub repository.
Any change pushed to this repo auto-publishes to the live site.

## To update the site
The website is generated from the design project. To change menu items,
prices, photos or copy, update the design and export a fresh `index.html`,
then commit it here — Cloudflare redeploys automatically.
